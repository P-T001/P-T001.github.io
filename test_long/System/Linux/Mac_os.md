快捷键（苹果键盘使用括号里面）

```
Win(command)+shift+3 #截全屏
Win(command)+shift+4 #手动截图到粘贴板
Win(command)+shift+5 #截图录屏工具界面，可以同时截屏和录屏，（先设置保存路径）

Win(command)+backspace（delete）#将选中文件放到回收站
Win(command)+shift+backspace（delete）#清空回收站

Win(command)+c                 # 复制
Win(command)+v                 # 粘贴
Win(command)+a                 # 全选
Win(command)+q                 # 关闭当前程序
Win(command)+w                 # 关闭当前窗口

```

命令

```
md5 文件路径   # 校验文件md5
```
