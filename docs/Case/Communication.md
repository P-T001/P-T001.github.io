## QQ

```
位置路径：C:/User/administrator(用户)/Tencent Files/QQ帐号  或  我的文档/Tencent Files/QQ帐号

FileRecv  接收文件

Image    收到的图片



导聊天记录:

直接导出QQ聊天记录mht格式，mht文件小能用浏览器直接打开

mht文件<200M，使用mht2html进行转换，该工具转换后的图片加载不了

mht文件>200M，使用QQMhtToHtml转换，html会切割成几个
```

## 微信

```
位置路径：C:/User/administrator(用户)/WeChat Files/微信帐号  或  我的文档/WeChat Files/微信帐号

config/accInfo.dat   用16进制编辑器打开，可以看到

下面都是以......分隔

开头是乱码，默认微信名，微信名称，0省，2市，4个性签名，f图像（大），h图像（小），绑定邮箱，绑定手机号

目录结构：

Attachment  缓存的图片

Backup  备份数据的文件

BackupFiles  备份文件

config   配置信息

CustomEmotions   表情图片，无后缀

CustomEmoV1     表情文件，被加密，新版才有

data          数据

FavTemp        图片缩略图，和小视频

Files 存放接收的文件，文档、xlsx表

HDHeadImage  存放高清的图片

Image/Image   缓存的图片
```
