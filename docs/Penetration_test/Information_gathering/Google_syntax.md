# Google_syntax

---

搜索语法

```
- 排除
+ 必须
inurl:/xxx/index.php  //网站存在的相对路径
intitle: 标题         //搜索标题
intext: 网页内容      //搜索网页内容
site: xxx.com         //搜索站点限制在这个域名范围
inurl:xxx             //搜索网站相对路径中含xx
link:xxx.com          //搜索含xxx.com的链接的网站页面
cache:xxx.com         //搜索出xxx.com网页历史快照，格式可能是乱的
filetype:xlsx         //搜索出有xlsx的网站路径
related:xxx.com        //搜索xxx.com同类型的网站，比如google.com可能会出现bing和百度的链接
```
