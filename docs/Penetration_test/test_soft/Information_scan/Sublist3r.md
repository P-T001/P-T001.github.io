# Sublist3r

---

下载：https://github.com/aboul3la/Sublist3r

功能：子域名挖掘

python sublist3r.py -d xxx.com

```
-d	- 领域	用于枚举子域的域名
-b	--蛮力	启用 subbrute bruteforce 模块
-p	--端口	针对特定的 tcp 端口扫描找到的子域
-v	--详细	启用详细模式并实时显示结果
-t	--线程	用于 subbrute bruteforce 的线程数
-e	- 引擎	指定以逗号分隔的搜索引擎列表
-o	- 输出	将结果保存到文本文件
-H	- 帮助	显示帮助信息并退出
```
